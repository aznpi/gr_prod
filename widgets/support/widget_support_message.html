What is Global Relay Message?

Designed for business and built for compliance, Global Relay Message provides counter-party authentication, guaranteed delivery, read receipts and FINRA, GDPR & MiFID2 compliance as part of a unified cloud-based service. Regulated organizations no longer need to manage disparate systems to satisfy IM recordkeeping requirements because Global Relay Message fuses its entire technology stack into one reliable compliant solution.

What devices can run it?

You can run Global Relay Message on the following supported devices:

  • Windows: Version 7 or later
  • macOS: Version 10.12 (Sierra) or later
  • Linux: Ubuntu 16.04 or later
  • iOS: Version 10.0 or later
  • Android: Version 5.1 or later
