Public IM

What public IM platforms and versions does Global Relay Archive support?

You must ensure your organization is using a supported IM client. We recommend you disable employees’ ability to upgrade their IM clients. This prevents users from installing newer, unsupported versions without your knowledge and ensures the settings for IM clients are not modified.

How does Public IM archiving work?
  1. Your organization configures its public IM clients to redirect messages through Global Relay’s proxy server. You have three options to configure routing:
    • Configure HOSTS File: Because you must configure the HOSTS file on each individual computer, this configuration works best for organizations with public IM clients on a small number of computers.
    • Create a DNA ‘A’ Record: 'A' record creation allows for central administration of message routing. It is a good option for organizations with public IM clients on a large number of computers.
    • IM client configuration: Yahoo and AIM clients allow users to change the server address the client connects to when a user logs in. This is the required method for AIM.
What is a HOSTS file and why do I need to edit this?
A HOSTS file is like an address book for your computer services. If you access a website like www.globalrelay.com, your computer will first look in the HOSTS file for the IP address. If the IP address is not in the HOSTS file, your computer will obtain it from your Internet Service Provider’s DNS server. Editing your HOSTS file directs public IM messages through Global Relay’s proxy server, allowing a copy of each message to be forwarded to Global Relay Archive.
Can I turn off the disclaimer message?
No. For privacy and legal purposes, your organization should disclose that IM messages are being archived.
Can I change the disclaimer message?
Yes. Appointed Administrators and IT contacts can contact our Support team at 866.484.6630 or support@globalrelay.net to change the disclaimer message.
I am being prompted to upgrade my IM client. Should I proceed?

No. You should not upgrade your IM client unless:

  1. Your IT team has instructed you to do so; AND
  2. The version you are upgrading to is included in the supported version list above.
II have recently upgraded my IM client. Should I do anything else?
Yes. We recommend you verify that the IM client is still routing messages to Global Relay. Contact your IT team if you are unsure.
Is there a report that documents public IM activity?
Yes. If you have been assigned an appropriate role, you can sign up for an automated report that documents public IM activity. Please contact our Training Team at 866.484.6630 for more information.
Where can I view my archived public IM messages?
Log in to Global Relay Archive and select "Instant Messages" from the Folder list on the main search page. Once you conduct a search, you will see instant messages on the results page. Global Relay Archive classifies each public IM platform with a unique icon.
